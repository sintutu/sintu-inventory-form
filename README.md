# Inventory Form Assignments

This repo is practice for Web Automation Advance in ndosiautomation sourced [here](https://ndosiautomation.co.za/#practice (Web Automation Advance contains the assignment at this link)).

There are two forms in the [assignment directory](./assignments):

* Short-form is available [here](./assignments/short-form-assignment.md).
* Long-form is available [here](./assignments/long-form-assignment.md).

## Build and run

Clone the repo and run tests from repo root with 

```shell
mvn -f ./shop-basic/pom.xml test
```

## Read more

Apply Google formatting styles with

```shell
mvn -f ./shop-basic/pom.xml com.spotify.fmt:fmt-maven-plugin:format
```

This is to be implemented in pipeline at a later stage.
